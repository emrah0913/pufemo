Veritabanı Erişim Hatası!

Bu panelin çalışması için Firebase veritabanı kurallarınızı güncellemeniz gerekiyor. Bu, genellikle yeni projelerde karşılaşılan bir durumdur.

Çözüm: Lütfen Firebase projenizin "Firestore Database" > "Rules" (Kurallar) sekmesine gidin. Mevcut kuralları silip aşağıdaki kuralları yapıştırın ve "Publish" (Yayınla) butonuna tıklayın.

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      // DİKKAT: Bu kural test aşaması içindir ve herkese okuma/yazma izni verir.
      // Canlıya geçmeden önce güvenlik kurallarını sıkılaştırmanız gerekir.
      allow read, write: if true;
    }
  }
}
